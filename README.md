# Java Data Structures & Management Systems

Bu repo, bilgisayar mühendisliği eğitimim kapsamında geliştirdiğim temel veri yapıları ve algoritma uygulamalarını içermektedir.

## 🚀 İçerik

### 1. Custom Generic Linked List
- **Teknoloji:** Java Generics, Iterator Pattern
- **Özellikler:** Sıfırdan dinamik bellek yönetimi mantığıyla kurgulanmış, her türlü veri tipiyle uyumlu bağlı liste yapısı.
- **Yetenekler:** `insertAt`, `addFirst`, `indexOf` ve `toArray` gibi temel metodların manuel implementasyonu.

### 2. Priority-Based Hospital System
- **Teknoloji:** Min-Heap, Queue, Java
- **Açıklama:** Hastaların kritiklik seviyelerine göre önceliklendirildiği bir acil servis simülasyonu.
- **Algoritma:** En acil hastaya $O(1)$ sürede erişim sağlayan Min-Heap yapısı kullanılmıştır.

## 🛠️ Nasıl Çalıştırılır?
Dosyaları klonladıktan sonra herhangi bir Java IDE'si (IntelliJ, Eclipse, VS Code) ile `App.java` veya `HospitalSystem.java` dosyalarını çalıştırabilirsiniz.
